## Khi nen dung tung skill

### `vercel-react-native-skills`

Nen dung khi:

- sua man hinh mobile phuc tap
- toi uu scroll/list performance
- lam viec voi safe area, keyboard, image, animation, gesture
- review pattern Expo/React Native

Khong can goi skill nay cho:

- task chi sua docs
- task chi sua service, adapter, store khong lien quan UI

### `vercel-composition-patterns`

Nen dung khi:

- refactor component API
- tach component lon
- xu ly prop drilling, boolean prop proliferation
- thiet ke reusable UI API ro hon

Khong can goi skill nay cho:

- bugfix nho khong doi component contract
- task API, route, auth hoac data layer

## Ket luan nhanh

- `vercel-react-native-skills` la skill ngoai huu ich nhat cho repo nay
- `vercel-composition-patterns` huu ich khi task nghieng ve component architecture
- Khong nen mo skill ngoai mac dinh cho moi task
