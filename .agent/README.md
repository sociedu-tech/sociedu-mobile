# .agent

## Tai lieu chinh

- `instruction.md`: luat lam viec, quy trinh lap trinh, nguyen tac ky thuat, quy chuan code.
- `mandatory-reading.md`: checklist cac file bat buoc phai doc truoc khi viet code theo tung loai task.
